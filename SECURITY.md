# Security Policy

## 公开仓库边界

本仓库只公开可复用的参考架构、匿名示例和模板。真实部署环境、网站内容、人员身份和凭据不属于本仓库。

## 禁止提交

- 真实 `.env`、Samba `users.conf` 或交接包；
- Makers API Token、GitHub Token、SSH 私钥或证书私钥；
- 网站内容、Docker 卷、数据库、备份和日志；
- 真实内网 IP、主机名、域名、个人主目录和未脱敏的服务器清单；
- 真实同事姓名、账号标识、交接路径或可关联到具体组织的内部信息。

## 发现泄露后的处理顺序

1. 停止继续推送或分享仓库；
2. 立即联系内部 IT 管理员；
3. 先撤销或轮换受影响的密码、Token 或密钥；
4. 检查 Git 历史、fork、clone、Actions 日志和制品；
5. 完成历史清理和再次扫描后再恢复协作。

不要通过公开 Issue、Pull Request 或聊天粘贴真实秘密。
